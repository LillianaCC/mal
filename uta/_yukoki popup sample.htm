THEME BLOGROLL #1
preview dark version & clear version
download dark version & clear version

This is my first blogroll theme. A minimalist, inspired in the Chrome page. I’m very satisfied with it and I hope you like it!

How to chance the colors?
I made it as simple as possible. Then you can search for “background:#282828” to change the darker color and “background:#2e2e2e” to change the lighter color of the dark version. To change in the clear version, look for “background:#f8f8f8” to change the darker color and “background:white” to change the color lighter. To change the color of the text/links, search for “color:#8f8f8f” in both versions.

Como mudar as cores?
Eu fiz o mais simples possível. Então você pode procurar por “background:#282828” para mudar a cor mais escura e “background:#2e2e2e” para mudar a cor mais clara da versão dark. Para mudar na versão clear, procure por “background:#f8f8f8” para mudar a cor mais escura e “background:white” para mudar a cor mais clara. Para alterar a cor do texto/links, procure por “color:#8f8f8f” em ambas as versões.
